# 📊 Budget vs Actual Sales Analysis Dashboard (FY2018)

## Overview

This project aims to analyze sales performance by comparing **Budget Sales** and **Actual Sales** across product sub-categories during FY2018.

The dashboard was developed using **Microsoft Excel** to help management monitor sales target achievement, identify performance gaps, and support data-driven decision making through variance analysis.

---

## Business Problem

Management needs a clear and efficient way to evaluate whether sales targets have been achieved across different product sub-categories.

Without a Budget vs Actual comparison, it is difficult to:

* Measure target achievement
* Identify high-performing and underperforming sub-categories
* Monitor sales performance regularly
* Support business decision making

This project addresses these challenges through an interactive Budget vs Actual Analysis Dashboard.

## Budget Planning Methodology

Since the dataset does not provide company sales targets, a Budget Sales table was created for each Quarter and Sub Category.

### Metrics Used

#### Actual Sales

Actual revenue generated from sales transactions.

#### Budget Sales

Target sales values assigned to each Sub Category.

#### Variance

Variance = Actual Sales − Budget Sales

#### Variance %

Variance % = (Actual Sales − Budget Sales) ÷ Budget Sales
